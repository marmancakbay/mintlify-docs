---
title: "List Users"
api: "GET https://api.deha.one/api/v1/users/{user_id}"
description: "List all users for your user."
---

Returns a list of all user profiles for your user, including their roles, permissions, and contact information.

## Authentication

<ParamField header="Authorization" type="string" required>
  Bearer token. `Bearer {your_token}`
</ParamField>
<ParamField header="X-User-Id" type="string" required>
  Your user identifier.
</ParamField>

## Path Parameters

<ParamField path="user_id" type="string" required>
  Your user identifier.
</ParamField>

## Response

<ResponseField name="users" type="array">
  Array of user profile objects.
</ResponseField>
<ResponseField name="total" type="integer">
  Total number of users.
</ResponseField>

## User Profile Object

| Field | Type | Description |
|-------|------|-------------|
| `user_id` | string | Unique user identifier |
| `email` | string | User email address |
| `display_name` | string | Display name |
| `role` | string | User role: `owner`, `admin`, `supervisor`, `agent`, or `viewer` |
| `config_permissions` | array | Configuration permissions: `read`, `write`, `deploy`, `test` |
| `allowed_agents` | array | List of agents the user can access |
| `allowed_channels` | array | List of channels the user can access |
| `created_at` | string | ISO 8601 creation timestamp |

## Error Responses

| Status | Description |
|--------|-------------|
| 401 | Missing or invalid authentication |
| 403 | Insufficient permissions |
