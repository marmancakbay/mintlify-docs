---
title: "List Tasks"
api: "GET https://api.deha.one/analytics/tasks"
description: "List all supported analysis tasks and their requirements."
---

Returns a list of all supported ML analysis tasks, including which engines support each task and whether a target column is required.

## Authentication

<ParamField header="Authorization" type="string" required>
  Bearer token. `Bearer {your_token}`
</ParamField>
<ParamField header="X-User-Id" type="string" required>
  Your user identifier.
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  Always `true` for this endpoint.
</ResponseField>
<ResponseField name="data.tasks" type="array">
  Array of task objects, each containing `name`, `engine` (supported engines), and `requires_target` (whether a target column is needed).
</ResponseField>

## Supported Tasks

| Task | Engines | Target Required | Description |
|------|---------|----------------|-------------|
| `classification` | auto, tabpfn, lightgbm | Yes | Predict categorical labels |
| `regression` | auto, tabpfn, lightgbm | Yes | Predict continuous values |
| `clustering` | statistical | No | Group data into clusters |
| `anomaly_detection` | statistical | No | Detect outliers and anomalies |
| `forecasting` | lightgbm | Yes | Time series prediction |
| `trend_analysis` | statistical | Yes | Linear trend detection |
| `correlation` | statistical | No | Feature correlation analysis |
| `summarization` | statistical | No | Descriptive statistics summary |
