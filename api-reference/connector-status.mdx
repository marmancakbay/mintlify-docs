---
title: "Connector Status"
api: "GET https://api.deha.one/connectors"
description: "List all connectors and their sync status for your user."
---

Returns a list of all configured connectors for your user, including their type, sync mode, enabled status, and last sync information.

## Authentication

<ParamField header="Authorization" type="string" required>
  Bearer token. `Bearer {your_token}`
</ParamField>
<ParamField header="X-User-Id" type="string" required>
  Your user identifier.
</ParamField>

## Query Parameters

<ParamField query="user_id" type="string" required>
  Your user identifier.
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  Whether the request completed successfully.
</ResponseField>
<ResponseField name="data.connectors" type="array">
  Array of connector objects.
</ResponseField>

## Connector Object

| Field | Type | Description |
|-------|------|-------------|
| `name` | string | Connector name |
| `connector_type` | string | Type: `rest_api`, `postgresql`, `mysql`, `file_storage`, `csv`, `excel`, `webhook` |
| `sync_mode` | string | Sync mode: `full_refresh`, `incremental`, or `cdc` |
| `enabled` | boolean | Whether the connector is active |
| `schedule_ref` | string | Reference to the associated schedule (if any) |
| `schema_ref` | string | Reference to the associated schema definition (if any) |
| `last_sync_at` | string | ISO 8601 timestamp of last sync |
| `last_sync_records` | integer | Number of records in the last sync |
| `last_status` | string | Status of last sync: `success` or `error` |
| `last_error` | string | Error message from last sync (null if successful) |

## Error Responses

| Status | Description |
|--------|-------------|
| 401 | Missing or invalid authentication |
