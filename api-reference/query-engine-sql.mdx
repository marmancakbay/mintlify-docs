---
title: "Direct SQL"
api: "POST https://api.deha.one/query/sql"
description: "Execute a validated SQL query directly against the data warehouse."
---

Executes a direct SQL query against your data warehouse after security validation. Only `SELECT` and `WITH` statements are permitted. Use this endpoint when you already know the SQL you want to run, rather than asking a natural language question.

## Authentication

<ParamField header="Authorization" type="string" required>
  Bearer token. `Bearer {your_token}`
</ParamField>
<ParamField header="X-User-Id" type="string" required>
  Your user identifier.
</ParamField>

## Request Body

<ParamField body="user_id" type="string" required>
  Your user identifier.
</ParamField>
<ParamField body="sql" type="string" required>
  SQL query to execute. Only `SELECT` and `WITH` (CTE) statements are allowed.
</ParamField>
<ParamField body="limit" type="integer" default="100">
  Maximum number of rows to return. Maximum allowed value is 10,000.
</ParamField>

## Response

Same response format as [Ask Question](/api-reference/query-engine-ask), with `query_mode` set to `direct_sql`.

## Error Responses

| Status | Description |
|--------|-------------|
| 400 | Non-SELECT/WITH statement or invalid SQL |
| 401 | Missing or invalid authentication |
| 500 | Query execution error |

<CodeGroup>
```bash cURL
curl -X POST "https://api.deha.one/query/sql" \
  -H "Authorization: Bearer eyJhbGciOiJSUzI1NiIs..." \
  -H "X-User-Id: acme_corp" \
  -H "Content-Type: application/json" \
  -d '{
    "user_id": "acme_corp",
    "sql": "SELECT COUNT(*) as total FROM orders",
    "limit": 100
  }'
```

```python Python
import httpx

response = httpx.post(
    "https://api.deha.one/query/sql",
    headers={
        "Authorization": f"Bearer {jwt_token}",
        "X-User-Id": "acme_corp"
    },
    json={
        "user_id": "acme_corp",
        "sql": "SELECT COUNT(*) as total FROM orders"
    }
)
```

```javascript Node.js
const response = await fetch(
  "https://api.deha.one/query/sql",
  {
    method: "POST",

      "Content-Type": "application/json"
    },
    body: JSON.stringify({
      user_id: "acme_corp",
      sql: "SELECT COUNT(*) as total FROM orders"
    })
  }
);
```
