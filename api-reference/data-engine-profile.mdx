---
title: "Profile Data"
api: "POST https://api.deha.one/data/profile"
description: "Generate a comprehensive statistical profile for a table."
---

Generates a detailed statistical profile of a table, including per-column statistics, data type inference, null counts, unique counts, outlier detection, and an overall data quality score.

## Authentication

<ParamField header="Authorization" type="string" required>
  Bearer token. `Bearer {your_token}`
</ParamField>
<ParamField header="X-User-Id" type="string" required>
  Your user identifier.
</ParamField>

## Request Body

<ParamField body="user_id" type="string" required>
  Your user identifier.
</ParamField>
<ParamField body="table_name" type="string" required>
  Name of the table to profile. Maximum of 50,000 rows will be profiled.
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  Whether the profiling completed successfully.
</ResponseField>
<ResponseField name="data.entity" type="string">
  Name of the profiled table.
</ResponseField>
<ResponseField name="data.total_records" type="integer">
  Total row count in the table.
</ResponseField>
<ResponseField name="data.total_columns" type="integer">
  Total number of columns.
</ResponseField>
<ResponseField name="data.field_profiles" type="array">
  Per-column profiles containing type information, null counts, unique counts, statistics, and outlier data.
</ResponseField>
<ResponseField name="data.quality_score" type="number">
  Data quality score (0 to 100), representing the completeness ratio as a percentage.
</ResponseField>
<ResponseField name="data.data_types_summary" type="object">
  Summary of data types across all columns (e.g., `{"integer": 3, "float": 2, "text": 5}`).
</ResponseField>
<ResponseField name="data.missing_summary" type="object">
  Summary of missing values including `total_missing_cells`, `total_cells`, and `completeness_ratio`.
</ResponseField>
<ResponseField name="data.duration_ms" type="number">
  Profiling duration in milliseconds.
</ResponseField>

## Field Profile Object

Each entry in `field_profiles` contains:

| Field | Type | Description |
|-------|------|-------------|
| `name` | string | Column name |
| `dtype` | string | Raw data type |
| `inferred_type` | string | Inferred type: `integer`, `float`, `boolean`, `datetime`, `datetime_string`, `numeric_string`, or `text` |
| `total_count` | integer | Total values |
| `non_null_count` | integer | Non-null values |
| `missing_count` | integer | Missing values |
| `missing_ratio` | number | Ratio of missing values |
| `unique_count` | integer | Distinct values |
| `uniqueness_ratio` | number | Ratio of unique values |
| `stats` | object | Numeric statistics: `mean`, `median`, `std`, `min`, `max`, `q25`, `q75`, `zeros_count`, `negative_count` |
| `outlier_count` | integer | Number of outliers (IQR-based) |
| `outlier_ratio` | number | Ratio of outliers |
| `top_values` | object | Top 10 most frequent values |
| `avg_length` | number | Average string length (text columns) |
| `max_length` | integer | Maximum string length (text columns) |
| `min_length` | integer | Minimum string length (text columns) |

## Error Responses

| Status | Description |
|--------|-------------|
| 401 | Missing or invalid authentication |
| 404 | Table not found |

  -H "Content-Type: application/json" \

)
