---
title: "Collections"
api: "GET /api/v1/collections"
description: "List, create, and manage vector collections for your knowledge base."
---

Manage vector database collections. Collections provide isolated namespaces for organizing documents within your user. Each collection stores vectors with a specific dimensionality and distance metric.

## Authentication

Requires JWT authentication via the proxy endpoint.

---

## List Collections

`GET /api/v1/collections`

Returns all collections belonging to your user with statistics.

<ParamField query="user_id" type="string" required>
  Your user identifier.
</ParamField>

### Response

<ResponseField name="user_id" type="string">
  Your user identifier.
</ResponseField>
<ResponseField name="collections" type="array">
  Array of collection objects, each containing:
  - `name` (string) -- Collection name
  - `status` (string) -- Collection health status (`green`, `yellow`, `red`)
  - `vectors_count` (integer) -- Number of vectors stored
  - `segments_count` (integer) -- Number of storage segments
  - `config` (object) -- Collection configuration including vector size and distance metric
</ResponseField>

<CodeGroup>
```bash cURL
curl "https://api.deha.one/proxy/acme_corp/vector/api/v1/collections?user_id=acme_corp" \
  -H "Authorization: Bearer eyJhbGciOiJSUzI1NiIs..."
```

```python Python
import httpx

response = httpx.get(
    "https://api.deha.one/proxy/acme_corp/vector/api/v1/collections",
    headers={"Authorization": f"Bearer {jwt_token}"},
    params={"user_id": "acme_corp"}
)
collections = response.json()
for col in collections["collections"]:
    print(f"{col['name']}: {col['vectors_count']} vectors ({col['status']})")
```

```javascript Node.js
const response = await fetch(
  "https://api.deha.one/proxy/acme_corp/vector/api/v1/collections?user_id=acme_corp",
  { headers: { "Authorization": `Bearer ${jwtToken}` } }
);
const data = await response.json();
```
</CodeGroup>


## Create Collection

`POST /api/v1/collections`

Creates a new collection for storing vectors. The collection name is automatically prefixed with your `user_id` if not already prefixed.

<ParamField body="user_id" type="string" required>
  Your user identifier.
</ParamField>
<ParamField body="collection_name" type="string" required>
  Name for the new collection. Will be prefixed with `{user_id}_` automatically if needed.
</ParamField>
<ParamField body="dimension" type="integer" required>
  Vector dimensionality. Must match the output dimension of your embedding model:
  - `1536` -- OpenAI `text-embedding-3-small`
  - `3072` -- OpenAI `text-embedding-3-large`
  - `768` -- Google Gemini / many HuggingFace models
</ParamField>

<CodeGroup>
```bash cURL
curl -X POST "https://api.deha.one/proxy/acme_corp/vector/api/v1/collections" \
  -H "Authorization: Bearer eyJhbGciOiJSUzI1NiIs..." \
  -H "Content-Type: application/json" \
  -d '{
    "user_id": "acme_corp",
    "collection_name": "products",
    "dimension": 1536
  }'
```

```python Python
import httpx

response = httpx.post(
    "https://api.deha.one/proxy/acme_corp/vector/api/v1/collections",
    headers={"Authorization": f"Bearer {jwt_token}"},
    json={
        "user_id": "acme_corp",
        "collection_name": "products",
        "dimension": 1536
    }
)
```

```javascript Node.js
const response = await fetch(
  "https://api.deha.one/proxy/acme_corp/vector/api/v1/collections",
  {
    method: "POST",
    headers: {
      "Authorization": `Bearer ${jwtToken}`,
      "Content-Type": "application/json"
    },
    body: JSON.stringify({
      user_id: "acme_corp",
      collection_name: "products",
      dimension: 1536
    })
  }
);
```
</CodeGroup>

```json Example Response
{
  "status": "created",
  "name": "acme_corp_products"
}
```

## Get Collection Stats

`GET /api/v1/collections/{name}/stats`

Returns detailed statistics for a specific collection.

<ParamField path="name" type="string" required>
  The full collection name (including user prefix, e.g., `acme_corp_knowledge`).
</ParamField>
<ParamField query="user_id" type="string" required>
  Your user identifier. Used for access control validation.
</ParamField>

### Response

<ResponseField name="name" type="string">
  Collection name.
</ResponseField>
<ResponseField name="status" type="string">
  Collection health status (`green`, `yellow`, `red`).
</ResponseField>
<ResponseField name="vectors_count" type="integer">
  Total number of vectors stored.
</ResponseField>
<ResponseField name="segments_count" type="integer">
  Number of storage segments.
</ResponseField>
<ResponseField name="config" type="object">
  Collection configuration including vector `size` and `distance` metric.
</ResponseField>

### Error Responses

| Status | Description |
|--------|-------------|
| `403` | Collection does not belong to your user |

```json Example Response
{
  "name": "acme_corp_knowledge",
  "status": "green",
  "vectors_count": 5000,
  "segments_count": 3,
  "config": { "size": 1536, "distance": "Cosine" }
}
```

## Delete Collection

`DELETE /api/v1/collections/{name}`

Permanently deletes a specific collection and all its vectors.

<ParamField path="name" type="string" required>
  The full collection name (including user prefix).
</ParamField>
<ParamField query="user_id" type="string" required>
  Your user identifier. Used for access control validation.
</ParamField>

<Warning>
This operation is irreversible. All vectors in the collection will be permanently deleted.
</Warning>

### Error Responses

| Status | Description |
|--------|-------------|
| `403` | Collection does not belong to your user |
| `404` | Collection not found |

<CodeGroup>
```bash cURL
curl -X DELETE "https://api.deha.one/proxy/acme_corp/vector/api/v1/collections/acme_corp_products?user_id=acme_corp" \
  -H "Authorization: Bearer eyJhbGciOiJSUzI1NiIs..."
```

```python Python
import httpx

response = httpx.delete(
    "https://api.deha.one/proxy/acme_corp/vector/api/v1/collections/acme_corp_products",
    headers={"Authorization": f"Bearer {jwt_token}"},
    params={"user_id": "acme_corp"}
)
```

```javascript Node.js
const response = await fetch(
  "https://api.deha.one/proxy/acme_corp/vector/api/v1/collections/acme_corp_products?user_id=acme_corp",
  {
    method: "DELETE",
    headers: { "Authorization": `Bearer ${jwtToken}` }
  }
);
```
</CodeGroup>

```json Example Response
{
  "status": "deleted",
  "name": "acme_corp_products"
}
```
